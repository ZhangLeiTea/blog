# markdown使用方法介绍

## 1. 代码块缩进

- 缩进例子
  - 一级缩进
    - 二级缩进，携带代码块

      ``` c++
      int a = 0;
      int b = 0;
      ```

    - 上面缩进的要素
      - 代码块前后要有空行
      - 代码块，开始的位置要同【二级缩进，携带代码块】这个标题（暂且叫标题）对齐

## 2. 列表的内容换行

  - 二级缩进，携带多行内容
